## Alt-Droid for F-Droid

```
  This is a repository of apps to be used with F-Droid. Applications
  in this repository are either official binaries built by the
  original application developers, or are binaries built from source
  by the admin using the tools on https://gitlab.com/fdroid
```

If you're interested in building a similar repo, there's a [rough guide](https://github.com/alt-droid/alt-droid-foss/wiki/How-To-Create-an-F-Droid-Repo).
